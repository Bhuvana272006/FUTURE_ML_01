#  Sales Forecasting ML App
##  Project Overview
This project is a Machine Learning-based Sales Forecasting application.
It predicts sales based on input features like quantity, discount, profit, and date.

## Model Used

* Random Forest Regressor
* Built using Scikit-learn
  
##  Features Used

* Quantity
* Discount
* Profit
* Year
* Month
* Day
* Day of Week
* Is Weekend
* Quarter

## Tech Stack

* Python
* Scikit-learn
* Pandas
* NumPy
* Streamlit

##  Live App

  Streamlit app link here   https://stores-sales-forecasting.streamlit.app/

## ▶️ How to Run Locally

1. Clone the repository
2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```

##  Output

The app takes user input and predicts the sales value using a trained ML model.

##  Conclusion

This project demonstrates an end-to-end ML workflow including:

* Data preprocessing
* Feature engineering
* Model training
* Deployment using Streamlit
